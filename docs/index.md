# Conductor

Conductor combines LLMs, APIs, and local actions into YAML workflows.

## What Can I Build?

- **Code review automation** - Analyze pull requests and provide feedback
- **Issue triage** - Label and route GitHub issues automatically
- **Daily digests** - Summarize activity from multiple sources
- **Slack/Discord bots** - Respond to messages with intelligent actions
