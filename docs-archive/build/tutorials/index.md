# Tutorials

Step-by-step guides for building real applications with ConductorSDK.

## Available Tutorials

### [Chat with File Access](./chat-application/)
Build a chat assistant that can read and write files using built-in actions.

**You'll learn:**
- Built-in actions with `WithBuiltinActions()`
- Agent loops with `RunAgent()`
- Real-time token streaming

### [Code Review Bot](./code-review-bot/)
Create a multi-step workflow that analyzes code and generates structured feedback.

**You'll learn:**
- Multi-step workflow construction
- Passing data between steps
- Structured output parsing

### [YAML Workflows](./yaml-workflows/)
Load and extend platform YAML workflows programmatically.

**You'll learn:**
- Loading YAML workflow files
- Combining YAML with programmatic steps
- Workflow validation

## Prerequisites

All tutorials assume you've completed the [Quick Start](../quickstart/) guide and have:
- Go 1.22+ installed
- An Anthropic API key (or alternative provider configured)
- Basic familiarity with Go
